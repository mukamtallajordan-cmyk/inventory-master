# 🚀 StockMaster Premium

**StockMaster** is a high-end, responsive inventory management system built with **React 18**, **TypeScript**, and **Vite**. It features a stunning **Glassmorphism** interface, real-time analytics, and comprehensive stock tracking capabilities.

![StockMaster Dashboard](file:///C:/Users/LENOVO/.gemini/antigravity/brain/f2303d47-93a7-4e02-8933-5c1df2f2f8d0/dashboard_with_charts_1774801475545.png)

## ✨ Features

- 📊 **Analytical Dashboard**: Visualize stock health with interactive charts (Recharts).
- 🔐 **Admin Access**: Secure login system with session management.
- 📦 **Inventory Management**: Full CRUD for products, categories, and suppliers.
- 🔄 **Stock Adjustments**: Detailed IN/OUT movement tracking with reasons and references.
- 📱 **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop.
- 🖨️ **Reports & Export**: Generate and export reports in **PDF** and **CSV**.
- 🔍 **Barcode Simulation**: Built-in barcode/QR scanner simulation for quick search.
- 💰 **Multi-currency Support**: Dynamic currency switching (EUR, USD, CFA, etc.).

## 🛠️ Tech Stack

- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Variables)
- **Icons**: Lucide-React
- **Animations**: Framer Motion
- **Persistence**: Browser LocalStorage

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mukamtallajordan-cmyk/inventory-master.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
